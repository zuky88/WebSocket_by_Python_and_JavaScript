# WebSocket by Python and javaScript

The server receives data from clients forever. 

The server is made by Python.

Clients are made by Python or JavaScript.

# Installation
```bash
pip install websockets
```

# Usage
Server
```bash
python3 server.py
```

Client by Python
```bash
python3 client.py
```

Clint by JavaScript (exampe)
```bash
 firefox index.htm
```
